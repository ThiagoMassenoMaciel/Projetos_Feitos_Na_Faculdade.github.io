7. O que é uma Lista? 

	lembre-se , cada posição desta array dinamica não tem o elemnto/objeto atribuido , tem a referência dele
	
	Um grupo de elementos em uma ordem linear, sequencialmente com antecessores e sucessores.
	pode repitir elementos
	
	para cada elemento tem um indice , começando com 0 
	
	este grupo é redimensinavel e ele pode ter elementos de tipo de dados diferente e tipo de dados especifico.
	
	funções

				isEmpty()
				add()

				indexOf()
