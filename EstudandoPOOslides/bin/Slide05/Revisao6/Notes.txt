6. O que é um mapeamento ou tabela ?

	é uma coleção de objetos que é mais eficiente manipular-los
	
	È uma estrutura de dados em que cada elemento uma chave que mapeia um valor. E estes dois podem ser objetos
	A chave não pode ser repetida , mas o valor pode. 
	Cada chave so pode mapear um valor
	
	Serve para acessar um elemento dentro de um grupo de elementos aleatoriamente através da chave
	
						
				
	Em programação orientada a objetos, um mapeamento ou tabela é uma estrutura de dados que armazena informações sobre objetos. Em outras palavras, um mapeamento é uma coleção de objetos que permite que os objetos sejam acessados e manipulados de forma eficiente. Em Java, por exemplo, um mapeamento pode ser criado usando a classe HashMap ou a classe TreeMap. 
	Para adicionar elementos a um mapeamento, você pode usar o método put(). Para remover elementos, você pode usar o método remove()1.
